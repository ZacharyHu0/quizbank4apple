# Quiz Challenge iOS App

一个基于SwiftUI的现代iOS测验应用，帮助用户学习和复习知识题库。

## 📱 功能特性

### 🏠 主页面
- **继续挑战** - 继续之前未完成的挑战
- **新挑战** - 选择题库开始新的学习挑战
- **选择题库** - 浏览和选择可用的题库
- **查看错题** - 复习答错的题目
- **历史统计** - 查看学习进度和统计数据
- **主题切换** - 支持明亮、深色和跟随系统主题

### 🎯 挑战页面
- **题目展示** - 清晰的题目和选项展示
- **进度指示器** - 可视化显示答题进度
- **实时统计** - 显示正确率和已答题数
- **导航控制** - 支持上一题、下一题和"不知道"选项
- **答案反馈** - 选择答案后立即显示正确/错误状态

### 📊 错题复习
- **错题列表** - 显示所有答错的题目
- **筛选排序** - 按时间、频率或题库排序
- **详细展开** - 查看完整题目和答案解析
- **搜索功能** - 快速搜索特定错题

### 📈 历史统计
- **学习概览** - 总挑战数、题目数、正确率等
- **表现趋势** - 图表显示学习进步趋势
- **题库分析** - 各题库的表现分析
- **最近活动** - 最近的挑战记录
- **时间筛选** - 支持按时间范围查看统计

## 🎨 设计特色

### Liquid Glass UI
- **毛玻璃效果** - 现代化的半透明界面
- **动态主题** - 自动适应系统主题变化
- **流畅动画** - 细腻的交互动画效果
- **Apple设计规范** - 遵循Apple Human Interface Guidelines

### 用户体验
- **直观导航** - 清晰的页面结构和导航逻辑
- **响应式设计** - 适配不同屏幕尺寸
- **无障碍支持** - 支持VoiceOver和其他辅助功能
- **离线使用** - 所有数据本地存储

## 🛠️ 技术栈

- **SwiftUI** - 现代化的UI框架
- **Combine** - 响应式编程

- **Charts** - 数据可视化（iOS 16+）

## 📖 使用说明

### 1. 开始使用
1. 打开应用，进入主页面
2. 点击"新挑战"选择题库
3. 开始答题挑战

### 2. 答题挑战
1. 阅读题目和选项
2. 点击选择答案
3. 使用导航按钮切换题目
4. 完成所有题目或选择"不知道"

### 3. 复习错题
1. 在主页面点击"查看错题"
2. 筛选和搜索错题
3. 点击展开查看详细答案

### 4. 查看统计
1. 在主页面点击"历史统计"
2. 查看各种学习数据
3. 分析学习进度和趋势

## 📝 CSV题库格式

CSV文件应包含以下列：

| 列名 | 说明 | 示例 |
|------|------|------|
| 题号 | 题目编号 | 1 |
| 年份 | 题目年份 | 23 |
| 模块 | 模块编号 | 5 |
| 周 | 周次 | 1 |
| 题干 | 题目内容 | Swift中的可选类型是什么？ |
| 选项 | 选项列表 | "A.字符串类型", "B.可选值类型", "C.数组类型", "D.函数类型" |
| 答案 | 正确答案 | B.可选值类型 |

## 🚀 构建和运行

1. 在Xcode中打开项目
2. 选择目标设备或模拟器
3. 点击运行按钮构建和运行应用
